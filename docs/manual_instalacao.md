
# Manual de Instalação

Bem-vindo ao guia de instalação do **Infarma HUB**.

## Requisitos Mínimos
* Windows 10 ou Superior
* 4GB de RAM
* Acesso à internet

## Passo a Passo
1. Baixe o instalador na tela inicial.
2. Execute como **Administrador**.
3. Siga as instruções na tela.

> **Nota:** Caso ocorra erro de permissão, contate o suporte.

### Comandos Úteis
Caso precise reiniciar o serviço manualmente:
```bash
net stop vmd_service
net start vmd_service
